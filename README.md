# Camera Detection MVP

Windows のブラウザでカメラ映像を開き、物体検出の枠と英語ラベルを重ねて表示する MVP です。  
COCO-SSD をブラウザ上で実行し、特定ラベルの簡易追尾、クリックロック、スナップショット保存までを含みます。

## 概要

このアプリは、ブラウザのカメラ入力を使ってリアルタイムに物体を検出し、以下のような機能を提供します。

- カメラ映像上への検出枠と英語ラベルの表示
- 特定ラベルだけを対象にした簡易追尾
- クリックした対象へのロック追尾
- ロック対象の視覚的強調
- オーバーレイ込みスナップショット保存
- 直近スナップショットのギャラリー表示
- 操作イベントのログ表示

## 起動方法

`getUserMedia()` を安定して使うため、`file://` ではなく `localhost` 経由で開いてください。

PowerShell で次を実行します。

```powershell
cd C:\Users\moshi\MyProjects\RaspberryPi\camera01
py -m http.server 8000
```

その後、ブラウザで次を開きます。

```text
http://localhost:8000
```

初回起動時はカメラアクセスの許可が必要です。

## 使い方

### 1. カメラを起動する

ページを開くと COCO-SSD モデルが読み込まれ、カメラ映像の上に検出結果が表示されます。

### 2. 物体を検出する

検出された物体には四角い枠と英語ラベルが表示されます。  
右側の `Detected Objects` に現在表示中の候補が一覧されます。

### 3. ミラー表示を切り替える

`Mirror preview` をオンにすると、プレビュー映像を左右反転できます。  
オーバーレイ座標もそれに追従します。

### 4. 特定ラベルだけを追う

`Track label` で対象ラベルを選ぶと、そのラベルだけを優先表示します。  
同じラベルが複数ある場合は、直前の位置に最も近い候補を継続採用します。

### 5. 誤検出を減らす

`Minimum confidence` を上げると、信頼度の低い検出を除外できます。  
迷ったら `55%` から `65%` あたりで調整してください。

### 6. 対象をクリックしてロックする

映像上の対象をクリックすると、その物体にロックします。  
ロック中は以下のように強調表示されます。

- 強調色の太い枠
- グロー
- 角マーカー
- 中心マーカー
- `LOCKED` ラベル

ロックを解除するには `Clear lock` を押します。

### 7. スナップショットを保存する

`Save snapshot` を押すと、現在の映像とオーバーレイを合成した PNG を保存します。  
保存画像には以下の情報が焼き込まれます。

- 対象ラベル
- 保存時刻

保存後は `Recent Snapshots` に直近の画像が表示されます。

### 8. 操作履歴を確認する

`Event Log` に以下のような操作履歴が表示されます。

- モデル読み込み
- カメラ起動
- ロック
- ロック解除
- 信頼度変更
- スナップショット保存

## 実装上のポイント

- 検出モデルは `TensorFlow.js` の `coco-ssd`
- カメラ取得は `navigator.mediaDevices.getUserMedia()`
- 表示は `video` + `canvas` のオーバーレイ構成
- `object-fit: cover` による表示座標のずれを補正
- ミラー表示時の座標変換にも対応
- 簡易追尾では、ラベル一致と前回位置に近い候補を優先
- 枠の揺れを抑えるために平滑化を実施

## 現時点の制約

- 物体検出カテゴリは COCO-SSD の学習済みラベルに依存します
- 厳密な物体追跡ではなく、検出ベースの簡易追尾です
- 同一ラベルの物体が多い場面では取り違えることがあります
- 検出速度と精度は PC 性能に依存します
- ブラウザ上のカメラアクセス許可が必要です
